Опис турніру

Tech Challenge — це онлайн-турнір з розробки програмного забезпечення, у якому команди створюють веб-рішення за заданим технічним завданням у обмежений час. Учасники повинні реалізувати функціональний продукт, використовуючи сучасні технології фронтенду, бекенду та баз даних. Особлива увага приділяється якості коду, архітектурі, зручності інтерфейсу та відповідності вимогам завдання. Роботи оцінює журі за технічною реалізацією, функціональністю та загальною якістю продукту. Більш детально можна ознайомитися з умовами завдання у файлі umova.pdf

Дозволені технології

Frontend: React, Vue.js, Angular, Svelte або інші сучасні JavaScript-фреймворки; також допускається використання чистого HTML, CSS та JavaScript. Backend: Node.js (Express, NestJS), Python (Django, Flask, FastAPI), Java (Spring), PHP (Laravel) або інші серверні технології з підтримкою ООП. База даних: MySQL, PostgreSQL, MongoDB, SQLite або інші реляційні чи NoSQL бази даних із обґрунтованою структурою. Контроль версій: Git з публічним репозиторієм на GitHub або GitLab. Деплой (опціонально):Vercel, Netlify, Render, Heroku, Railway, або інші платформи для демонстрації роботи проєкту
